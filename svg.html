SVG

  • SVG is a vector based graphics in XML format
  • SVG stands for Scalable Vector Graphics
  • SVG is used to define graphics for the web page
  • SVG has several methods for drawing paths, boxes, circles, text and graphic images

uses of SVG

  • SVGs can be edited in any text editor
  • SVGs are scalable, however jpg, jpeg, png are not scalable. If the image is scalable then we can increase or decrease the image size. If it is not scalable we cannot increase or decrease the image size, the image gets worse
  • SVGs are zoomable
  • SVG can be printed with High Quality at any resolution
  • SVG can be search ,indexed, scripted and compressed
  • SVG files are pure XML. So we can use it in html and css
